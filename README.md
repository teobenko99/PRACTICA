# Portfolio de Análisis de Datos – Teo Benko

Este repositorio contiene trabajos prácticos y proyectos de análisis de datos desarrollados en el marco de mi formación en Ingeniería Industrial y Especialización en Ciencia de Datos.

El foco de los trabajos está puesto en el análisis exploratorio, limpieza de datos, construcción de variables y modelado, con una mirada aplicada a la toma de decisiones de negocio.

## Contenidos del repositorio

Los notebooks siguen una lógica de pipeline analítico:

### 1. Exploración y previsualización de datos
- Análisis inicial de datasets
- Identificación de variables relevantes
- Detección de valores faltantes y outliers

### 2. Limpieza y preparación de datos
- Tratamiento de valores nulos
- Normalización y transformación de variables
- Preparación del dataset para análisis y modelado

### 3. Feature engineering
- Creación de nuevas variables
- Selección de atributos relevantes
- Preparación de datasets finales

### 4. Modelado
- Implementación de modelos de aprendizaje automático
- Evaluación de resultados
- Análisis de métricas

## Herramientas utilizadas
- Python (Jupyter Notebook)
- Análisis exploratorio de datos
- Modelado predictivo

## Enfoque
Si bien los datasets utilizados son académicos, los análisis están orientados a entender el comportamiento de los datos y generar conclusiones aplicables a contextos reales de negocio, como operaciones, finanzas y gestión de clientes.

 Este repositorio forma parte de mi portfolio profesional.
